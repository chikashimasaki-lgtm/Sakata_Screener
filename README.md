# 酒田五法 シグナル・スクリーナー

株価API（Yahoo Finance 日足）から多銘柄をスキャンし、**酒田五法のシグナルが点灯した銘柄を一覧化**する
GAS ＋ GitHub Pages アプリ。**投資助言ではなく、シグナル抽出の補助ツール。**

## 検出パターン

| パターン | 内容 | 方向 |
|---|---|---|
| 赤三兵 | 陽線3本の切り上げ | 買い |
| 三羽烏(黒三兵) | 陰線3本の切り下げ | 売り |
| 三空踏み上げ | 上の窓が3連続（買われ過ぎ） | 売り |
| 三空叩き込み | 下の窓が3連続（売られ過ぎ） | 買い |
| 上げ三法 | 長陽→値幅内の調整→上抜け陽線 | 買い |
| 下げ三法 | 長陰→値幅内の調整→下抜け陰線 | 売り |

| 三山(三尊天井) | 3つの山（中央が最高）でネックライン割れ | 売り |
| 三川(逆三尊) | 3つの谷（中央が最安）でネックライン上抜け | 買い |

| 明けの明星 | 長大陰線→窓開けの星→陽線でC実体中心を回復 | 買い |
| 宵の明星 | 長大陽線→窓開けの星→陰線でC実体中心を割る | 売り |
| 捨て子線(明け/宵) | 中央が同事線で両側に窓が開く強い反転（明け=買い/宵=売り） | 買い/売り |

### 現代版フィルター（RSI補強・close値のみ）

形状シグナルにRSI(14)の裏付けを加点し、だましを抑える。

- **三空踏み上げ** … RSI80超なら `RSI過熱(80超)` を加点（反落を補強）
- **三空叩き込み** … RSI20割れなら `RSI底値(20割れ)` を加点（反発を補強）
- **三山/三川** … 天底でRSIが逆行（ダイバージェンス）なら `RSIダイバージェンス` を加点

※ 出来高フィルター・ボリンジャーのバンドウォークは、日足barsに出来高が無く容易に反映できないため未対応。

## セットアップ

1. Apps Script プロジェクトを作成し `src/` を push（clasp）
2. スプレッドシートを再読み込み → メニュー「酒田五法」→ セットアップ
3. 「銘柄」シートにコード(4桁)を入力
   - または「プライム銘柄を取得（J-Quants）」で自動取得（要 `JQUANTS_API_KEY` スクリプトプロパティ）
4. 「シグナル走査/続行」を実行（銘柄数が多いと時間分割で自動再開）
5. 「JSON出力」で `sakata_signals.json` を作成 → `index.html` の隣に置くと公開ページが実データ表示

## GitHub Pages

`index.html` は `sakata_signals.json` を読み込み、シグナル一覧（検索・方向/パターン絞り込み・ソート）を表示。
JSON が無ければサンプルで描画。Settings → Pages → `main` / `/(root)` で公開。

## 注意

- データは Yahoo Finance 日足（`.T` 東証）。シグナルはだましもあるため必ず自身で確認すること。
- 銘柄数が多い場合、走査は GAS の実行時間内で分割し 90 秒後に自動再開する。
